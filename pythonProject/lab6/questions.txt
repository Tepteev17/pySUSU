1.for <переменная> in <последовательность>:
    <Блок>
else:
    <Блок>
Семантика:
-Перебираются все элементы
Последовательности по порядку и для каждого
из них выполняется Блок. 
-На каждой итерации цикла Идентификатору
присваивается значение текущего элемента
последовательности.

2.while <Условие>:
    <Блок> 
else:
    <Блок>

Семантика:
-Блок операторов тела цикла while выполняется пока
Условие истинно.
-Если Условие ложно – выполняется Блок, следующий за
else.

4\5. __name__ и __main__ - Инструкции. __name__ хранит название программы.
__main__ указывает на область видимости, где будет выполняться код. 
Если запустить Python-файл прямо, то значением __name__ будет __main__. 
Если же его запустить в качестве модуля, то значением будет уже не __main__, а название модуля.
Это значит, что строка выше вернет True только в том случае, если программа будет запущена прямо.
Если же ссылаться на файл как на модуль, то содержимое конструкции if не будет выполнено.

6. Открыть файл ::= open(‘<путь>‘[, ‘<режим>‘] [,’<буферизация>’]), где:
-путь – относительный или абсолютный путь к файлу
-режим – по умолчанию ‘r’ - открытие на чтение (является значением по умолчанию)
-также можно поставить режим 'w' - открытие на запись, содержимое файла удаляется, если файла не существует, создается новый.

7. Если мы с помощью функции open попробуем открыть несуществующий файл, то:
-если режим будет 'w', то создастся новый файл.
-иначе будет ошибка.
